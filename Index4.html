Projeto Tinkercad


Circuito feito em Tinkercad para a disciplina LIP em q é feito um programa híbrido (Automação e Múltiplos Arduinos computação distribuída)codificado utilizando linguagem de blocos e linguagem alfanumérica. Nesse produto cada membro da equipe será o líder do projeto e desenvolvimento de uma parte do sistema. Para conceitos necessários alguns links podem ser visitados como:


Foi usado como modelo base o projeto do Carlos Augusto , professor de Lip da Puc Minas link do seguinte tutotialhttps://all3dp.com/2/tinkercad-arduino-how-to-design-simulate-circuits/

O grupo do projeto possui:
Filipe Arthur: O circuito y é um hacking de um projeto que simula uma porta automática que abre dada uma senha correta. Neste projeto foi adicionada uma opção para alterar a senha.

Lucas Diniz - O circuito 2 realiza o acionamento do buzzer com push buttom com 2 arduinos se comunicando por um fio.

E eu: no meu código através de um sensor ultrassônico, que calcula a distância entre o sensor e o objeto mais próximo. Se a distância for maior que 113, o motor servo gira em um sentido, se for menor gira no outro sentido e caso a distancia seja 0 um led piscará. E sempre essa distancia será mostrada em um LCD
